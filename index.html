Fourier Series

In this section we shall go through Fourier Series.

What is Fourier Series

A fourier series is an expansion of a periodic function f(x) in terms of an infinite sum of sines and cosines of a fundamental angle and its harmonics.
The Fourier Series of a harmonic function f(x) with a period 2c defined in the interval [-c:c] is usually expressed as:
$$f(x) = \frac{a_0}{2}+ \sum_{n=1}^{\infty} a_n cos(\frac{n \pi x}{c})+ \sum_{n=1}^{\infty} b_n sin(\frac{n \pi x}{c})$$ where \(a_n\) and \(b_n \) are constants for all positive integer values of n. And these constants are: $$a_0 = \frac{1}{c}\int_{-c}^{c} f(x) dx$$ $$a_n = \frac{1}{c} \int_{-c}^{c} f(x) cos(\frac{n \pi x}{c}) dx$$ $$b_n = \frac{1}{c} \int_{-c}^{c} f(x) sin(\frac{n \pi x}{c}) dx$$ Thus the coefficients can be determined by evaluating the above mentioned integeral.
A function can be expressed in Fourier series if it satisfies the Dirichlet conditions. Which are,

  1. The function is single valued and bounded
  2. The function has finite number of discontinuities and maxima and minima.
  3. f(x+2c) = f(c) for all values of x outside [-c:c]

program boson
integer::ne,np,i,j,s,k
real,allocatable::energy(:),e_config(:)
real::part_func,tem
open(20,file = 'part_boson.dat',status='unknown')
ne = 3
np = 3
allocate(energy(ne),e_config((np+1)**ne-1))
energy = (/2.,3.,4./)
call energy_config(ne,np,energy,e_config,s)
tem = 0.1
do while(tem<100.0)
call Partition(e_config,s,tem,part_func)
write(20,*)tem,part_func
tem = tem + 0.05
enddo
end program boson

subroutine Partition(e_config,s,tem,part_func)
integer::s,k
real::part_func,e_config(s)
part_func = 0.0
do k = 1,s
part_func = part_func + exp(-1.0*e_config(k)/tem)
enddo
end subroutine Partition

subroutine energy_config(ne,np,energy,e_config,s)
integer::ne,np,j,s,i
integer::binary(ne)
real::energy(ne),e_config((np+1)**ne-1)
s = 0
do j = 1,(np+1)**ne-1
call dec2nin(j,ne,np+1,binary)
if (sum(binary) == np) then
s = s+1
e_config(s) = dot_product(binary,energy)
print*,(binary(i),i=1,ne),e_config(s)
endif
enddo
return
end subroutine energy_config

subroutine dec2nin(decgg,bit,x,bin)
integer::bit,x     ! generate x nary number(x=2 for binary)
integer::bin(bit),dec,decgg
dec = decgg
do i = bit,1,-1
bin(i) = mod(dec,x)
dec = dec/x
enddo
dec = decgg
end subroutine dec2nin
